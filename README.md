## Getting Started

1. Fork this repository and clone it into the current directory

   ```bash
   git clone https://github.com/viniciusmoreeira/github-profiler.git
   ```

2. Install dependencies

   ```bash
   yarn
   ```

3. Fire up the server and watch files

   ```bash
   yarn start
   ```
